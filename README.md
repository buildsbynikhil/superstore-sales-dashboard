# Superstore Sales Dashboard

## Overview
This project is an interactive Excel Sales Dashboard built using the Superstore dataset.

## Features
- Sales Analysis
- Profit Analysis
- Region-wise Analysis
- Category-wise Analysis
- Interactive Slicers
- Pivot Tables
- Pivot Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Files

- superstore-sales-dashboard.xlsx - Excel workbook containing the dataset and interactive dashboard
- dashboard-preview.png
- Dashboard preview
