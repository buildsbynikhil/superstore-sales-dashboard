# Superstore Sales Dashboard

## Overview
This project is an interactive Excel Sales Dashboard created using the Superstore dataset.

## Features
- Sales Analysis
- Profit Analysis
- Category-wise Performance
- Region-wise Performance
- Interactive Slicers
- Pivot Tables
- Pivot Charts

## Tools Used
-Microsoft Excel
-Pivot Tables
-Pivot Charts
-Slicers
